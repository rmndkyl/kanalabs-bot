## Setup

1. Clone this repository:
   ```bash
   git clone https://github.com/rmndkyl/kanalabs-bot.git
   cd kanalabs-bot
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Setup: edit config.js to your needs and input privkey and addr there
   ```bash
   nano config.js
   ```
    ![config.js](image.png)
4. Run The Script:
   ```bash
   npm run start
   ```
